# 项目简介

## 核心功能
- 基于YOLOv8的减速带检测系统，支持图片、视频及摄像头进行检测
- 提供完整源码、3400张标注数据集、训练好的模型及训练结果
- 支持选择不同模型，自定义置信度和IOU阈值
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存功能
- 可选择显示标签和原图

## 适用场景
- 交通监控与安全系统
- 智能驾驶辅助系统
- 道路设施检测与维护
- 学术研究与教学演示
- 自动驾驶技术研发

## 技术特点
- 采用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 技术栈：Python3.9、opencv-python、PyQt5、ultralytics、torch1.9
- 内含3400张数据集，包含['Speed Bump']单类标注
- 提供UI界面源文件、图标及完整项目运行教程
- 支持多种输入源的检测功能，包括单张图片、图片文件夹、视频文件和本地摄像头
